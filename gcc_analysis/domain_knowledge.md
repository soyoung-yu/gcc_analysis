# 도메인 지식 문서

> 본 문서는 AI Agent가 분석 업무 수행 시 참고할 도메인 용어 및 데이터 구조를 정의합니다.
> 해당 도메인은 화장품 ODM 기업이며, 고객사 의뢰부터 연구, 개발, 확정, 초도 생산 과정까지 데이터에 대한 설명입니다.

---

## 1. 코드 체계 개요

### 코드 간 계층 구조

```
신제품접수코드 (P9xxx / B9xxx)
└── 벌크구분코드 (P3xxx / B3xxx)  [1개 이상]
    └── 벌크코드 (3xxx)           [1개 이상]
        └── 랩넘버 (실험)         [1개 이상 → 최종 1개 확정]

완제품코드 (9xxx)
└── 벌크코드 (3xxx)               [1개 이상]
└── 부자재                        [1개 이상]
```

---

## 2. 코드 정의

### 2-1. 신제품접수코드

- **정의**: 의뢰가 접수될 때 해당 의뢰 단위에 부여되는 코드. **의뢰 구분의 기본 단위**.
- **기본 Prefix**: `P9`
- **예외 Prefix**: 벌크 의뢰(NPD 의뢰 아닌 경우)는 `B9`, 일부 법인·상황에서는 다른 Prefix 사용 가능.
- **관계**: 하나의 신제품접수코드에 여러 개의 벌크구분코드가 속할 수 있음.

### 2-2. 벌크구분코드

- **정의**: 의뢰 내 벌크 단위를 구분하는 코드.
- **기본 Prefix**: `P3`
- **예외 Prefix**: 벌크 의뢰(NPD 의뢰 아닌 경우)는 `B3`, 일부 법인·상황에서는 다른 Prefix 사용 가능.
- **관계**: 하나의 벌크구분코드에 여러 개의 벌크코드(3%)가 확정될 수 있음.

> **공통 참고**: 신제품접수코드와 벌크구분코드를 통칭하여 **"NPD Code"** 라고 부르기도 함.

**예시**
```
P3K1260016501 : 이그니스 클렌징볼 시카
  ├── 3ENS00051110 : 이그니스오원시카효소클렌징볼(H&A파마켐가공)(상온)(CICA)
  ├── 3ENS00052110 : 이그니스오원쌀효소클렌징볼(H&A파마켐가공)(상온)(RICE)
  └── 3ENS00053110 : 이그니스오원팥효소클렌징볼(H&A파마켐가공)(상온)(PORE)
```

### 2-3. 벌크코드

- **정의**: 실제 처방(내용물 벌크) 하나에 대한 코드.
- **기본 Prefix**: `3`
- **예외 Prefix**: 일부 법인은 다른 Prefix 사용 가능.
- **관계**: 하나의 벌크코드는 여러 실험(랩넘버)을 거쳐 **최종 랩넘버 1개**로 확정됨.

### 2-4. 완제품코드

- **정의**: 판매 단위의 완성 제품 하나를 의미하는 코드.
- **기본 Prefix**: `9`
- **구성**: 벌크(내용물, 벌크코드) 1개 이상 + 부자재 포함.

### 2-5. 랩넘버

- **정의**: 벌크 개발 과정에서 수행되는 각 실험에 부여되는 번호.
- **관계**: 하나의 벌크코드는 여러 랩넘버를 가질 수 있으며, 최종적으로 1개의 랩넘버로 확정됨.

> **주의**: 법인별로 코드 Prefix 규칙이 다를 수 있음. 법인 컨텍스트를 항상 확인할 것.

## 4. 고객사 및 브랜드

- 하나의 고객사는 여러 브랜드를 보유할 수 있음.
- 하나의 고객사는 **법인별로 서로 다른 고객사코드**를 가질 수 있음.
  - 동일 고객사라도 법인이 다르면 코드가 다를 수 있으므로, 고객사 조회 시 법인 조건을 함께 확인해야 함.

---

## 5. 제형 코드

- **정의**: 제형을 구분하는 소유형 코드.
- **형식**: 숫자 2자리 + 문자 1자리 + 숫자 4자리 (총 7자리)
- **구조 예시**:

```
13S0302
││└──── 위치 3: 카테고리 문자 (S = 스킨케어, M = 메이크업, ...)
│└───── 위치 2: 숫자
└────── 위치 1: 숫자
        └──────────── 뒤 4자리: 세부 분류 숫자
```

| 위치 | 예시값 | 의미 |
|---|---|---|
| 3번째 자리 (문자) | `S` | 스킨케어 |
| 3번째 자리 (문자) | `M` | 메이크업 |

> 위 카테고리 문자 외에도 다른 값이 존재할 수 있으며, 각 위치별 세부 의미는 별도 코드 테이블 참조 필요.
> 위치 1, 2, 뒤 4자리의 경우 각자 위치 값에 따라 정해진 규칙이 존재합니다.

## 6. 주요 주의사항 (Agent 참고용)

| 항목 | 주의 내용 |
|---|---|
| Prefix 예외 | 법인 또는 의뢰 유형에 따라 기본 Prefix 규칙이 다를 수 있음. Prefix만으로 코드 유형을 단정하지 말 것. |
| 고객사코드 | 동일 고객사라도 법인별로 코드가 다를 수 있음. |
| 벌크 확정 | 벌크코드의 최종 확정은 랩넘버 기준이며, 확정 전 여러 후보 랩넘버가 존재할 수 있음. |
| NPD Code | 신제품접수코드 또는 벌크구분코드를 지칭하는 통칭으로 혼용될 수 있음. 문맥에 따라 구분 필요. |
| 완제품 ↔ 벌크 | 완제품 1개에 벌크 여러 개가 포함될 수 있음. 1:1 관계로 가정하지 말 것. |
