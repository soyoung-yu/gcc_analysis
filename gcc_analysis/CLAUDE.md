# CLAUDE.md — AI 작업 규칙

이 폴더에서 작업할 때 반드시 아래 규칙을 따른다.

---

## 규칙

### 1. 지시한 사항에 대한 코드 수정만 진행
- 요청받은 범위 외의 코드는 임의로 수정하거나 리팩토링하지 않는다.
- 관련 없는 정리, 추가 기능 구현, 스타일 변경은 하지 않는다.

### 2. 코드 수정 전 반드시 짧게 보고
- 어떤 내용을 왜 수정하는지 한 문장으로 먼저 설명한다.
- 수정 후 결과도 간단히 요약한다.

### 3. 모든 코드에 한 줄 주석 추가
- 함수, 딕셔너리, 주요 로직 블록마다 목적을 설명하는 한 줄 주석을 작성한다.
- 주석은 "무엇을 하는지"가 아니라 "왜 하는지 / 어떤 역할인지"를 중심으로 작성한다.
