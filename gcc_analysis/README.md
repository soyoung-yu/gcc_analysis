# GCC Analysis

법인별 수작업 데이터를 하나의 표준 데이터마트로 통합하는 프로젝트입니다.

---

## 파일 구성

### 스크립트

| 파일 | 역할 |
|---|---|
| `manual_to_standard_datamart.ipynb` | 메인 분석 노트북. 법인별 엑셀 데이터를 로드하고 표준 데이터마트로 변환하는 전체 로직을 포함 |

### 문서

| 파일 | 역할 |
|---|---|
| `manual_data_by_entity.md` | 법인별 수작업 데이터 구조 정의. 파일명, Raw Data 시트명, 헤더 행 위치, 집계 필터링 조건, 예외사항 등을 기술 |
| `manual_to_standard_data_by_entity.md` | 표준 데이터마트 CSV 산출물 설명. 생성 기준, 표준 컬럼 정의, 법인별 현황, 데이터 품질 이슈 포함 |
| `column_mapping_NPD.md` | 법인별 수작업 컬럼명 → 표준 데이터마트 컬럼명 매핑 정의. 혼합 사용 컬럼 및 복수 컬럼 합산 규칙 포함 |
| `domain_knowledge.md` | GCC 도메인 지식 정리. 코드 체계, 용어 정의, 업무 프로세스 등 분석에 필요한 배경 지식 |
