# 법인별 수작업 데이터 설명

> 각 법인은 의뢰(NPD), 확정, 초도, 매출 데이터를 각자의 방식으로 조회하여 제출하고 있음.
> 본 문서는 법인별 제출 데이터의 구조 및 집계 기준을 정의함.

---

## 공통 사항

- 데이터 유형: **NPD(의뢰)**, **확정**, **초도**, **매출** 4종
- NPD 데이터의 의뢰 기준 단위는 **벌크구분코드(P3%)** (법인별 예외 사항은 아래 참고)
- 파일 구성 방식은 법인마다 상이함 (파일 분리 vs 단일 파일 내 Sheet 분리)

---

## 데이터 위치
